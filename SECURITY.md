# Security Policy

## Supported Versions

This project is currently in active development. Security fixes are provided on `main`.

| Version | Supported |
| ------- | --------- |
| main    | Yes       |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately via email:
- `rajarshic@gmail.com`

Include the following when possible:
- Affected endpoint(s) and request/response samples
- Reproduction steps
- Impact assessment (confidentiality, integrity, availability)
- Suggested remediation (if available)

### Response Targets
- Acknowledgment: within 72 hours
- Initial triage: within 7 days
- Patch plan: within 14 days for high/critical issues

## Disclosure Policy

After a fix is available and deployed, we may publish a security advisory with:
- Impact summary
- Affected versions
- Mitigation and upgrade guidance
